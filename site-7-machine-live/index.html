What leaves the machine — and what never can.

Transparency cuts both ways, so here's ours. The extractor that feeds this page works on an allowlist: there is no code path that copies a sentence from an internal report to this site.

passes through: counts · filing times · traffic-light tallies
never passes: email content · names · clients · systems · anything with letters it didn't write itself
belt and braces: the output is deny-scanned for addresses, IDs, paths and names — one hit and it refuses to publish
when the feed is stale: this page says so instead of pretending
